# Instrukcja uruchamiania projektu

## Frontend
1. Zainstaluj potrzebne zależności:
   ```bash
   npm install
   ```
2. Uruchom serwer frontendowy:
   ```bash
   ng serve
   ```

## Backend
1. Zainstaluj potrzebne zależności:
   ```bash
   npm install
   ```
2. Uruchom serwer backendowy:
   ```bash
   node server.js
   ```
